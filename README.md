# Devvit Website

A community of builders & hackers. Built with React + Vite + Tailwind CSS.

## Stack
- **Framework**: React 18 + React Router
- **Bundler**: Vite
- **Styling**: Tailwind CSS + CSS variables
- **Fonts**: Lexend, Inter, Space Grotesk

## Brand Colors
| Token            | Hex       | Usage              |
|------------------|-----------|--------------------|
| `devvit-bg`      | `#0E0E0E` | Background         |
| `devvit-purple`  | `#A7A5FF` | Primary accent     |
| `devvit-orange`  | `#FF7439` | Secondary accent   |
| `devvit-white`   | `#FFFFFF` | Primary text       |

## Getting Started

```bash
npm install
npm run dev
```
