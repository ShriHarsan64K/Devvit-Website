# Devvit Style Guide

## Colors

| Name   | Hex       | Usage                        |
|--------|-----------|------------------------------|
| BG     | `#0E0E0E` | Page background              |
| Purple | `#A7A5FF` | Primary accent, CTA buttons  |
| Orange | `#FF7439` | Secondary accent, dates, CTA |
| White  | `#FFFFFF` | Primary text                 |

### Opacity Variants
- `rgba(255,255,255,0.6)` — Body text
- `rgba(255,255,255,0.4)` — Muted text
- `rgba(255,255,255,0.1)` — Borders
- `rgba(255,255,255,0.05)` — Subtle backgrounds

## Typography

### Fonts
| Font          | Weights         | Usage                        |
|---------------|-----------------|------------------------------|
| Lexend        | 400,700,800,900 | Display, headings, logo      |
| Inter         | 400,500,700     | Body text, paragraphs        |
| Space Grotesk | 400,500,700     | Labels, tags, nav, buttons   |

### Size Scale (fluid clamp)
| Use Case   | Value                          |
|------------|--------------------------------|
| Hero       | `clamp(3.5rem, 12vw, 10rem)`  |
| Page title | `clamp(3rem, 10vw, 7rem)`     |
| Section h2 | `clamp(2rem, 5vw, 4rem)`      |
| Card title | `clamp(1.25rem, 3vw, 1.5rem)` |
| Body       | `0.875rem` / `1rem`           |
| Label/Tag  | `0.7rem` — `0.8rem`           |

## Buttons

| Variant     | Background  | Text       | Use Case            |
|-------------|-------------|------------|---------------------|
| `primary`   | `#A7A5FF`   | `#0E0E0E`  | Main CTAs           |
| `secondary` | Transparent | `#FFFFFF`  | Secondary actions   |
| `orange`    | `#FF7439`   | `#FFFFFF`  | Community/join CTAs |
| `ghost`     | Transparent | `#A7A5FF`  | Subtle actions      |

## Responsive Breakpoints
| Name    | Width    |
|---------|----------|
| Mobile  | < 480px  |
| Mobile  | < 768px  |
| Tablet  | < 1024px |
| Desktop | 1024px+  |
